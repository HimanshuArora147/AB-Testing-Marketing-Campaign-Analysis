
# AB Testing: Marketing Campaigns

## 📌 Project Overview
This project analyzes 365 days of ad campaign data from **Facebook** and **AdWords** to determine which platform is more effective in terms of **click-through rate (CTR)**, **conversion rate**, and **cost per click (CPC)**. The goal is to recommend the most cost-effective platform and optimize marketing ROI.

## 🔍 Objective
- Identify which platform performs better using **A/B testing** and **statistical analysis**.
- Provide data-driven recommendations to improve **ROI**.
